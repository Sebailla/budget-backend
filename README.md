# Budget API

## Arquitectura:

MVC (Model View Controler)

## Tecnología:
1. Express: ```versión - 5.1.0```
2. Color
3. Morgan
4. Typescript
5. ORM: Sequelize 6
6. BD: PostgreSQL 16
7. bcryptjs
8. nodemailer
9. JWT

#
> [!IMPORTANT]
> 1. Clonar repositorio.
> 2. Renombrar `env.template` a `.env` y agregar sus variables de entorno.
> 3. Ejecutar: ```npm install```
> 4. Correr el servidor : ```npm run dev:api```
